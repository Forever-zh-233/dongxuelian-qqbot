## 三、任务 1：补全 6 个工具

### 3.1 工具清单与规格

每个工具放 `lib/agent/tools/`，按现有模板。完成后加到 `registry.js`。

#### append_file.js

```yaml

description: 追加内容到文件末尾，不覆盖已有内容。

  content: string (required) - 追加内容
```

- 返回写入字节数

#### grep_search.js

```yaml
name: grep_search
description: 在文件中搜索匹配的文本行，返回上下文。
dangerous: false
defaultChannels: ['dashboard']
params:
  pattern: string (required) - 搜索正则
  path: string (required) - 文件绝对路径
  contextLines: number (default 2) - 上下文行数
```
- 路径白名单
- 返回 `file:line: content` 格式
- 最多 200 行匹配结果

#### execute_javascript.js

```yaml
name: execute_javascript
description: 执行 JavaScript 代码片段。用于数据处理、格式转换。
dangerous: true
defaultChannels: ['dashboard']

  code: string (required) - JS 代码，用 return 返回结果
```
- `new Function()` + `'use strict'`
- 禁止 require/import/process/child_process/fs/global/__dirname/__filename
- 10s timeout
- 返回 JSON.stringify 结果

#### send_file_to_user.js

```yaml

dangerous: false
defaultChannels: ['qq']
```

- 检查文件存在 + 路径白名单
- 调用 OneBot WS `send_group_file` / `upload_file`
- 如果 OneBot 不可用返回提示

#### get_token_usage.js

```yaml

description: 获取当前 Agent 会话的 Token 使用统计。
dangerous: false
defaultChannels: ['dashboard', 'qq']
```

- 读取 stats.js 累计数据
- 返回 `{ estimatedTokens, totalCalls, byTool: {...} }`

#### set_user_timezone.js

```yaml

description: 设置当前用户的时区偏好。
```

## 六、任务 4：stats.js 充实

### 6.1 当前状态

基础计数：`recordCall()` + `getStats()`。

### 6.2 目标状态

```js
// 新增结构
let callRecords = []          // 最近 200 条详细记录
let tokenEstimates = []       // 每轮 token 消耗
let cumulativeTokens = 0     // 累计 token

function recordCall(toolName, channel, durationMs, tokenEstimate, success) {
  callRecords.unshift({ tool: toolName, channel, duration: durationMs, tokens: tokenEstimate, success, at: Date.now() })
  if (callRecords.length > 200) callRecords.length = 200
  cumulativeTokens += tokenEstimate
}

function getStats() {
  return {
    totalCalls: callRecords.length,
    totalTokens: cumulativeTokens,
    recentCalls: callRecords.slice(0, 20),
    byTool: groupBy(callRecords, 'tool'),
    byChannel: groupBy(callRecords, 'channel'),
    successRate: callRecords.filter(r => r.success).length / Math.max(1, callRecords.length),
    avgDurationMs: avg(callRecords.map(r => r.duration)),
    avgTokensPerCall: Math.ceil(cumulativeTokens / Math.max(1, callRecords.length)),
  }
}
```

### 8.1 改动点

三处代码修改，约 10 行：

**改法 1** — `chat.js` L443，时间那行从 `systemPrompt +=` 改为独立变量：

```js
// 原：
systemPrompt += '\n当前时间：' + now.getHours() + '时' + now.getMinutes() + '分。核心信息...'

// 改：
const dynamicTime = '当前时间：' + now.getHours() + '时' + now.getMinutes() + '分。核心信息...'
```

**改法 2** — `chat.js` L520-521，messages 拆分：

```js
// 原：
const messages = [{ role: 'system', content: systemPrompt }]

// 改：
const messages = [{ role: 'system', content: systemPrompt }]
messages.push({ role: 'system', content: dynamicTime })
```

**改法 3** — `chat.js` L801-807，合并加 provider 条件：

```js
// 原：无条件合并
for (let i = messages.length - 1; i >= 1; i--) { ... }

// 改：
const cfg = await loadConfig()
if (cfg.provider === 'dashscope') {
  for (let i = messages.length - 1; i >= 1; i--) {
    if (messages[i].role === 'system') {
      messages[0].content += '\n\n' + messages[i].content
      messages.splice(i, 1)
    }
  }
```
