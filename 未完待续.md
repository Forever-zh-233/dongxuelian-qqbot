# LianLianBot 未完待续

这份文档只记录未来路线。当前安装、部署、启动和维护入口以 `README.md` 为准。

## 插件市场与插件模板

- 建立可检索的插件市场，区分官方插件、实验插件和用户插件。
- 提供标准插件模板，覆盖配置声明、命令注册、日志、测试和发布说明。
- 给插件增加版本、依赖、权限和兼容性标记，方便 Dashboard 做安装前提示。

## Persona Studio

- 提供人格、世界观 lore、模式和触发规则的可视化编辑器。
- 支持人格预览、对话样例测试和变更历史。
- 增加导入/导出能力，让人格包可以独立分享和回滚。

## 部署健康中心

- 汇总 Node/npm、NapCat、Koishi、OneBot、端口、数据目录和日志状态。
- 给常见失败提供可操作诊断，例如端口占用、扫码未完成、依赖缺失、API Key 无效。
- 把 Windows 本地部署和远程 Linux 部署的健康检查统一到同一套状态模型。

## App 包与更新机制

- 将可重建程序文件、运行时文件、用户数据和日志进一步分层。
- 支持校验发布包完整性，减少下载损坏或半更新状态。
- 设计保守的更新流程：先备份关键配置，再替换程序文件，失败时能恢复到上一个可用版本。

## 备份、迁移与多实例

- 提供一键备份配置、人格、记忆、图集、黑白名单和日志摘要。
- 支持把一个实例迁移到新目录或新机器。
- 规划多实例管理，允许不同 QQ 号、不同群策略和不同数据目录并存。

## 可视化工作流

- 将定时任务、事件触发、Agent 工具调用和通知动作组合成可视化流程。
- 支持基础条件分支，例如群号、用户、关键词、时间段和运行状态。
- 给高风险动作增加确认、日志和回滚提示。

## 素材与卡片资产库

- 扩展莲莲图集，支持图片、音频、短视频、卡片模板和渲染素材统一管理。
- 为 A-G 闪卡样式提供可预览的模板库。
- 支持素材标签、来源备注、使用统计和失效检测。
