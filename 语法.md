# 昵称 / 集合语法

## 1. 绑定昵称

```text
@A用户 昵称 名称A
@A用户 昵称名称A
```

作用：

- 给 `A用户` 绑定昵称 `名称A`。
- 两种写法都支持。

## 2. 删除昵称

```text
删除昵称 名称A
删除昵称 名称A @A用户
```

作用：

- 不带 `@` 时，默认删除自己在该名称下的绑定。
- 带 `@` 时，删除指定成员绑定。

## 3. 查看昵称

```text
查看昵称 名称A
谁是 名称A
```

作用：

- 查看某个昵称绑定到谁。
- 如果该名称下有多个人，也会一起列出来。

## 4. 查看成员

```text
查看成员 A用户
查看成员 @A用户
```

作用：

- 反查某个成员有哪些昵称或属于哪些集合。

## 5. 查看全部昵称

```text
查看全部昵称
nicklist
```

作用：

- 只显示本群中“只有 1 人”的条目。
- 这些条目视为昵称，不是集合。

## 6. 创建集合

```text
创建集合 集合A @A用户 @B用户
```

作用：

- 新建一个集合。
- 集合里至少应有 2 人，才有“集合”的意义。

## 7. 查看集合

```text
查看集合 集合A
```

作用：

- 查看集合成员列表。

## 8. 查看全部集合

```text
查看全部集合
集合列表
```

作用：

- 只显示本群中“2 人及以上”的条目。
- 不会再把单人昵称混进去。

## 9. 集合添加

```text
集合添加 集合A @A用户 @B用户
```

作用：

- 向已有集合中继续加人。
- 已在集合中的成员不会重复加入。

## 10. 集合删除

```text
集合删除 集合A @A用户 @B用户
```

作用：

- 从集合中移除指定成员。

## 11. 清空集合

```text
清空集合 集合A
确认清空集合 集合A
```

作用：

- 两步确认，避免误操作。

## 12. 删除集合

```text
删除集合 集合A
确认删除集合 集合A
```

作用：

- 两步确认，彻底删除整个集合。

## 13. 重命名

```text
重命名昵称 名称A 名称B
重命名集合 集合A 集合B
```

作用：

- 把旧名称改成新名称。
- 目标名称已存在时不会覆盖。

## 14. 复制集合

```text
复制集合 集合A 集合B
```

作用：

- 按当前成员列表复制出一个新集合。

## 15. 合并集合

```text
合并集合 集合A 集合B
```

作用：

- 把 `集合B` 中的成员合并进 `集合A`。
- 已在 `集合A` 中的成员不会重复加入。
- 当前实现是“合并到目标集合”，不是“创建或合并集合 @成员”。

## 16. 集合运算

```text
集合交集 集合A 集合B
集合并集 集合A 集合B
集合差集 集合A 集合B
```

作用：

- 查看两个集合的交集、并集、差集结果。

## 17. 批量艾特

```text
at名称A
at集合A
```

作用：

- 把对应条目里的成员一次性 `@` 出来。
