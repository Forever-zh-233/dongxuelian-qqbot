# Bot 调用语法

更新时间：2026-04-20

## 功能一：B 站视频解析 / 发送

由 `vedio.md` 部署，插件名为 `local-video-sender`。

### 自动解析

在 QQ 群里发送以下任意内容，Bot 会自动解析并发送视频：

```text
https://www.bilibili.com/video/BVxxxx
BVxxxx
https://b23.tv/xxxx
B 站小程序分享卡片
包含 B 站链接的普通文本
```

### 说明

- 支持 B 站标准视频链接。
- 支持 `BV` 号。
- 支持 `b23.tv` 短链接。
- 支持 QQ 小程序分享卡片。
- 该功能和昵称集合功能是两个独立插件，互不影响。

## 功能二：群昵称 / 集合艾特

由 `name.md` 部署，插件名为 `group-name-at`。

### 基础绑定

```text
@某人 昵称 猪
```

作用：给被 @ 的成员绑定昵称 `猪`。

返回示例：

```text
已为 创纪元 添加昵称「猪」。
```

如果同一个昵称绑定到第二个人，会自动升级为集合：

```text
已将sulto_玲加入集合「猪」，已自动升级为集合，当前共 2 人。
```

第三个人及以后加入同名集合时，不再显示“已自动升级为集合”。

### 艾特昵称 / 集合

```text
at猪
```

作用：艾特昵称或集合里的所有成员。

返回示例：

```text
@创纪元@sulto_玲
```

说明：

- 返回只包含艾特，不追加 `（猪，共2人）` 这类尾巴。
- 实际存储按 `userId` 绑定，群友改群昵称不会导致绑定错乱。
- 查询和艾特时会尽量刷新成员群昵称。

### 删除单个昵称绑定

```text
删除昵称 猪
删除昵称 猪 @某人
```

作用：

- 不 @ 人时，默认删除发送者自己在 `猪` 下的绑定。
- @ 人时，删除被 @ 成员在 `猪` 下的绑定。

### 查看集合 / 昵称

```text
查看集合 猪
查看昵称 猪
谁是 猪
```

作用：查看 `猪` 下有哪些成员。

这三个命令等价。

### 查看全部集合

```text
查看全部集合
查看全部昵称
集合列表
nicklist
```

作用：列出本群所有昵称 / 集合。

这些命令等价。

### 查看某个成员的所有昵称 / 集合

```text
查看成员 创纪元
查看成员 @某人
@某人 昵称
```

作用：查看这个成员有哪些昵称，或在哪些集合里。

## 集合管理

### 创建集合

```text
创建集合 猪 @a @b @c
```

作用：一次性创建集合并批量加人。

规则：

- 如果 `猪` 不存在，则创建。
- 如果 `猪` 已存在，会提示已存在，避免误覆盖。
- 批量添加时自动去重。

### 合并集合

```text
合并集合 A B
```

作用：

- 把集合 `B` 的成员合并进集合 `A`。
- `A` 和 `B` 都必须已经存在。
- 已经在 `A` 里的成员不会重复添加。
- 合并后 `B` 仍然保留，不会被删除。

### 集合添加

```text
集合添加 猪 @a @b @c
```

作用：给已有集合批量加人。

规则：

- 集合不存在时会提示未找到。
- 自动去重。
- 如果从 1 人变 2 人，会提示自动升级为集合。
- 如果原本已经是 2 人以上，不再提示自动升级。

### 集合删除

```text
集合删除 猪 @a @b
```

作用：从集合里批量移除成员。

### 清空集合

```text
清空集合 猪
确认清空集合 猪
```

作用：保留集合名，清空所有成员。

安全规则：

1. 先发送 `清空集合 猪`。
2. Bot 返回确认提示。
3. 60 秒内再次发送 `确认清空集合 猪` 才会真正清空。

### 删除集合

```text
删除集合 猪
确认删除集合 猪
```

作用：删除整个集合。

安全规则：

1. 先发送 `删除集合 猪`。
2. Bot 返回确认提示。
3. 60 秒内再次发送 `确认删除集合 猪` 才会真正删除。

### 重命名集合 / 昵称

```text
重命名集合 猪 佩奇
重命名昵称 猪 佩奇
```

作用：把 `猪` 改名为 `佩奇`。

说明：

- 两个命令当前等价。
- 如果目标名 `佩奇` 已存在，不会覆盖。

### 复制集合

```text
复制集合 猪 狗
```

作用：复制 `猪` 的成员到新集合 `狗`。

规则：

- 原集合必须存在。
- 目标集合不能已存在，避免误覆盖。

## 集合运算

### 交集

```text
集合交集 A B
```

作用：列出同时在 `A` 和 `B` 里的成员。

### 并集

```text
集合并集 A B
```

作用：列出在 `A` 或 `B` 任意一个集合里的成员。

### 差集

```text
集合差集 A B
```

作用：列出在 `A` 里、但不在 `B` 里的成员。
