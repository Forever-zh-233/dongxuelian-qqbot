# 昵称 / 集合指令速查

## 帮助菜单

| 想做什么 | 指令 |
|---|---|
| 查看东雪莲总帮助 | `help东雪莲` / `帮助东雪莲` / `东雪莲help` |
| 查看集合帮助 | `help集合` / `帮助集合` |

## 昵称

| 想做什么 | 指令 |
|---|---|
| 绑定昵称 | `@A 昵称 名称A` / `@A 昵称名称A` |
| 删除自己的昵称 | `删除昵称 名称A` |
| 删除指定人的昵称 | `删除昵称 名称A @A` |
| 查昵称是谁 | `查看昵称 名称A` / `谁是 名称A` |
| 查某人有哪些昵称/集合 | `查看成员 @A` / `查看成员 A用户` |
| 查看全部昵称 | `查看全部昵称` / `nicklist` |

## 集合

| 想做什么 | 指令 |
|---|---|
| 创建集合 | `创建集合 集合A @A @B` |
| 查看集合 | `查看集合 集合A` |
| 查看全部集合 | `查看全部集合` / `集合列表` |
| 添加成员 | `集合添加 集合A @A @B` |
| 移除成员 | `集合删除 集合A @A @B` |
| 清空集合 | `清空集合 集合A` → `确认清空集合 集合A` |
| 删除集合 | `删除集合 集合A` → `确认删除集合 集合A` |
| 重命名集合 | `重命名集合 集合A 集合B` |
| 复制集合 | `复制集合 集合A 集合B` |
| 合并集合 | `合并集合 集合A 集合B`，把 B 合进 A |

## 集合运算

| 想做什么 | 指令 |
|---|---|
| 交集 | `集合交集 集合A 集合B` |
| 并集 | `集合并集 集合A 集合B` |
| 差集 | `集合差集 集合A 集合B`，输出 A-B |

## 批量艾特

| 想做什么 | 指令 |
|---|---|
| 艾特昵称/集合 | `at名称A` / `at集合A` |

## 记住这几条

- 昵称：1 人条目。
- 集合：2 人及以上条目。
- `查看全部昵称` 只看单人条目。
- `查看全部集合` / `集合列表` 只看多人条目。
- 删除、清空集合需要二次确认。
