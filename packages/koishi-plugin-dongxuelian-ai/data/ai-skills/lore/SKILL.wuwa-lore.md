# 《鸣潮》世界观完整知识库（剧情更新版，截止至3.3版本）

> 本文件仅陈述游戏内已公开的设定事实，不包含任何角色主观评价。AI 在回答时应按需提取信息，并用角色口吻自然表达。

---

## 一、星球与基础概念

- **索拉里斯（Solaris-3）**：故事发生的星球，名称源自科幻小说《索拉里斯星》，其版图与地球呈镜像对称。
- **悲鸣（Lament）**：周期性席卷索拉里斯的毁灭性大灾难，导致物理法则改变，万物皆由"频率"构成。
- **残象（Phantom）**：由异常频率形成的无固定形态怪物，是悲鸣的直接产物。
- **残象潮（Phantom Tide）**：残象大规模、高频率出现的灾难性事件。
- **无音区（Empty Realm）**：被悲鸣长期侵蚀的极度危险区域，常为禁地。
- **超频（Overclocking）**：共鸣者过度使用能力或受执念影响导致共鸣失控，伤害自身与周围。
- **共鸣者（Resonator）**：能够与特定频率共鸣并运用力量的人。
- **协奏能量（Concerto Energy）**：弥漫于天地间的能量粒子，战斗中协奏、变奏的能量基础。
- **黑石（Black Stone）**：蕴藏特殊能量的稀有矿物，被部分势力用作能源。
- **天空海（Sky Ocean）**：索拉里斯特有的倒悬异象海洋。
- **声骸（Echo）**：击败特定残象后收集其"残响"，通过终端转化为实用道具。黎那汐塔曾将其发展为公共技术。
- **虚质（Virtual Matter）**：拉海洛地区特有的异常物质，与地区和鸣式阿列夫一的影响密切相关。

## 二、主要地区与势力

### 1. 瑝珑（Huanglong）

- **设计原型**：中国盛唐 + 山水画。
- **体制**：一庭六州。今州是六州之一，为北方边塞军事重镇。
- **岁主**：角（Jue），龙形，龙尾上竖，代表"理性"与"秩序"。
- **关联人物**：长离（今州令尹参事）、今汐（角之共鸣者）。

### 2. 黎那汐塔（Rinascita）- 2.0-2.7篇已完结

- **别称**：声骸之国。
- **体制**：城邦制，主要城邦为拉古那、七丘。
- **主要宗教组织**：隐海修会，曾为岁主英白拉多的代理人，现任主座芬莱克已殁。
- **岁主与鸣式（最终结局）**：
  - **岁主英白拉多（老马）**：天空/天马形态，执掌"分离"权柄。在2.x版本剧情中已死亡，但其权能仍支撑着黎那汐塔声骸系统的运转。
  - **鸣式利维亚坦**：大海/鲸鱼形态，执掌"融合"权柄。在2.x版本大决战中被击败。其作为根植于文明和人类思想中的存在，**并未真正消亡，在结局时仍处于蛰伏状态**。
- **关联人物（结局状态）**：
  - **卡提希娅**：岁主与鸣式的共同共鸣者，作为"融合"与"分离"权柄的集合体，已在2.7迎来最终结局。
  - **坎特雷拉**：在牺牲自己、将力量交予老马后，作为"真圣女"确认已死亡。

**版本摘要**：黎那汐塔故事线已在2.7版本彻底完结。当前版本不应再出现黎那汐塔正在发生的动态剧情。

### 3. 拉海洛（Lahalo）- 3.x 主线核心

- **版本定位**：3.x 版本的主舞台，故事于 3.3 版本《自星海尽处回响》到达重要高潮，但并未完全完结。
- **地理位置**：位于索拉里斯之极，罗伊冰原正下方的地下巨型构造体。
- **背景与生态**：古时因地极变动成为新极点，原住民罗伊人迁入地下绿洲"拉海洛"。
- **核心势力**：
  - **深空联合**：在此创立星炬学院，是一座无国界的学园都市。
  - **残星会**：在拉海洛幕后活跃，试图利用鸣式的力量，计划通过回收黑石的副产物"永晖石"达成目标。
- **地区守护者（隧者）**：拉海洛地区拥有独立的守护者 **"隧者"**，是当地岁主级别的存在，掌管隧门，对抗鸣式阿列夫一。
- **当前鸣式威胁（阿列夫一）**：
  - **名称**：阿列夫一
  - **地位**：拉海洛地区的鸣式，是当前版本最大的威胁。
  - **能力**：拥有"时序错乱"的能力，能侵蚀现实，创造虚质空间。
  - **当前状态（3.3 结局）**：在3.3版本的高潮剧情中，漂泊者驾驶"隧者"机甲进入虚质空间，经过一场大战后，已将其**驱逐（而非彻底消灭）**到宇宙边缘，但危机并未根除。
- **关键地图与地点**：
  - **黯原**：位于拉海洛最深处，隧锚所在的地底空间。
  - **隧门**：连接现实与虚质空间的门扉。
  - **虚质空间**：被阿列夫一侵蚀的异空间。
- **关键人物**：
  - **爱弥斯**：曾被困于虚质空间，在3.3剧情中被成功营救。
  - **绯雪**：深空联合特别对策组唯一成员，被选中继承岁主"铃"的残片之力。
  - **达妮娅**：星炬学院的学生，真实身份复杂，受制于残星会。
- **新增玩法/机制**：
  - **隧者机甲**：漂泊者可以驾驶的巨型机甲，用于对抗鸣式。
  - **终端替换系统**：新增拉海洛终端，可替换之前的今州"葫芦"与黎那汐塔"提灯"。

### 4. 黑海岸（Black Shores）

- **性质**：隐藏于风暴海上的秘密组织，致力于研究并解决悲鸣。
- **组织架构**：首领称"守岸人"；下设"执花"（负责寻找有潜质的共鸣者）。已确认椿是其"执花"。
- **核心技术**：泰提斯系统（监测悲鸣）、黑石能源。
- **关联**：漂泊者是该组织的创立者之一。

### 5. 残星会（Fractsidus）

- **性质**：极端宗教与科研团体，全游戏主要敌对势力。
- **目标**：试图利用鸣式的力量，计划通过回收黑石的副产物"永晖石"达成目的。
- **领袖**：斯瓦茨洛。
- **近期行动（3.3 版本）**：在拉海洛地区活跃，计划利用绯雪曾经变成鸣式共鸣者的经历来掌握鸣式的力量。
